## 📋 Sobre o Projeto

Este é um simulado completo no estilo **Certo/Errado (C/E) do CEBRASPE** para o cargo de **Agente Administrativo da Polícia Federal (edital 2025)**. O simulado contém **120 questões** abrangendo todas as disciplinas do concurso, com base no caderno de provas oficial.

### 🎯 Disciplinas Abordadas

| Disciplina | Quantidade |
|------------|------------|
| Língua Portuguesa | 18 questões |
| Raciocínio Lógico | 6 questões |
| Estatística | 3 questões |
| Ética no Serviço Público | 5 questões |
| Lei 8.112/90 | 3 questões |
| Lei de Improbidade Administrativa | 4 questões |
| Informática | 9 questões |
| Gestão Pública | 20 questões |
| Licitações e Contratos | 6 questões |
| Logística | 4 questões |
| Segurança Privada | 10 questões |
| Produtos Químicos | 10 questões |
| Lei de Migração | 10 questões |
| Armas | 5 questões |
| Administração Pública | 7 questões |

## ✨ Funcionalidades

- ✅ **120 questões** no estilo CEBRASPE (Certo/Errado)
- ⏱️ **Timer de 70 segundos** por questão (simula tempo real de prova)
- 📊 **Barra de progresso** e contador (1/120, 2/120...)
- 🎯 **Feedback visual imediato** (verde para acerto, vermelho para erro)
- 🏆 **Classificação final** (Aprovado, Classificado, Habilitado)
- 💾 **Opção para salvar resultado** com nome
- 🔗 **Botão para ver gabarito completo** (link configurável)
- 📱 **Totalmente responsivo** (funciona perfeitamente em celulares)
- 🎨 **Cores institucionais** da PF (azul escuro e amarelo)
